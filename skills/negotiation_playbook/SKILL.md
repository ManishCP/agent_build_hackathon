# Negotiation playbook skill

## Instructions
1. Collect all 🔴 findings first, then 🟡
2. Rank: personal guarantee > NNN uncapped > assignment > holdover > auto-renewal > rest
3. One specific ask per item — no vague advice
4. Identify top 3 "must-get" changes
5. Estimate dollar value of each win where possible

## Gotchas
- Bundle all asks together — don't negotiate one at a time
- Every change must be in writing — verbal means nothing
- Negotiate before signaling you want the space
- Walk-away items: full PG, assignment prohibited, holdover >200%
